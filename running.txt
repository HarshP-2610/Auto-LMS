### How to Run the Project

Follow these steps to get the application up and running:

#### 1. Database Setup
- Ensure **MongoDB** is installed and running on your local machine.
- The default connection string is `mongodb://127.0.0.1:27017/autolms`.

#### 2. Backend Setup
- Open a terminal and navigate to the `backend` directory:
  ```powershell
  cd backend
  ```
- Install dependencies (if you haven't already):
  ```powershell
  npm install
  ```
- Start the backend server:
  ```powershell
  node server.js
  ```
- The backend will typically run on [http://localhost:5000](http://localhost:5000).

#### 3. Frontend Setup
- Open a **new terminal** and navigate to the root directory (`app` folder).
- Install dependencies (if you haven't already):
  ```powershell
  npm install
  ```
- Start the Vite development server:
  ```powershell
  npm run dev
  ```
- The frontend will typically run on [http://localhost:5173](http://localhost:5173).

---
**Note:** Make sure both the backend and frontend are running simultaneously in separate terminal windows.

#### 4. Default Admin Credentials
The system automatically seeds an admin account on the first run:
- **Email:** `admin@autolms.com`
- **Password:** `Admin1234`
